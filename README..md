## General Info

Documento creado para el curso de "Data Science" en Coderhouse. Este trabajo de Machine Learning tiene el objetivo de analizar una base de datos sobre reservas del Hotel Inn, con el fin de reducir la cantidad de reservas canceladas de forma online. <br>
